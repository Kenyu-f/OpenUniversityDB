## 🇺🇸 English

### 📖 About

This repository is a **long-term, structured research database** covering leading research universities worldwide. Unlike a general university guide, it is built specifically for **students, researchers, and developers** who want structured, sourced information on:

- Admissions processes (undergraduate & graduate)
- Research opportunities, laboratories, and faculty
- Funding, fellowships, and internships
- Alumni, spin-off companies, and technology transfer
- A roadmap for aspiring researchers

All content is written in Markdown, cites **official university sources**, and follows a single unified template so that every university can eventually be compared side by side.

### 🗂️ Repository Structure

```text
universities/
└── <university-slug>/
    ├── overview.md               # History, campus, divisions, rankings, stats
    ├── admissions.md             # UG & graduate admissions requirements
    ├── undergraduate_research.md # SURF, fellowships, independent research
    ├── graduate_research.md      # PhD/MS programs, advisor matching
    ├── laboratories.md           # Every lab: dept, focus, director, website
    ├── professors.md             # Current faculty directory
    ├── research_centers.md       # Institutes & centers
    ├── internships.md            # Internal/external internships
    ├── funding.md                # Scholarships, grants, assistantships
    ├── publications.md           # Repositories, thesis archives, OA policy
    ├── companies.md              # Startups, spin-offs, tech transfer
    ├── alumni.md                 # Notable alumni & honors
    ├── roadmap.md                # Path from high school to research career
    └── resources.md              # Official links, OCW, YouTube, podcasts
```

### 🚀 Goal

Expand this database to cover the **world's top 100 research universities**, using a consistent 15-file template per institution, so the repository can eventually support cross-university comparison and discovery.

### 🤝 Contributing

Contributions are welcome. Please:

1. Follow the existing 15-file template exactly (see `universities/caltech/` as the reference implementation).
2. Cite only official or highly reliable sources; flag any figure that could not be verified.
3. Keep tone objective, encyclopedic, and factual — no speculation.
4. Open a Pull Request with a clear description of which university/file you're adding or updating.

## 🇯🇵 日本語

### 📖 概要

このリポジトリは、世界の主要な研究大学を対象とした**長期運用を前提とした構造化リサーチデータベース**です。一般的な大学ガイドとは異なり、以下のような情報を**学生・研究者・開発者**向けに、体系的かつ出典付きで整理することを目的としています。

- 学部・大学院の出願プロセス
- 研究機会、研究室、教員情報
- 資金援助、フェローシップ、インターンシップ
- 卒業生、スピンオフ企業、技術移転
- 研究者を目指す人向けのロードマップ

すべてMarkdownで記述し、**大学公式ソース**を優先的に引用し、統一されたテンプレートに従うことで、将来的に大学同士を横断比較できる構成を目指しています。

### 🗂️ リポジトリ構成

```text
universities/
└── <university-slug>/
    ├── overview.md               # 歴史・キャンパス・学部構成・ランキング・統計
    ├── admissions.md             # 学部・大学院の出願要件
    ├── undergraduate_research.md # SURF、フェローシップ、独立研究
    ├── graduate_research.md      # 博士・修士課程、指導教員マッチング
    ├── laboratories.md           # 全研究室(学科・分野・責任者・公式サイト)
    ├── professors.md             # 現役教員一覧
    ├── research_centers.md       # 研究所・センター
    ├── internships.md            # 学内外インターンシップ
    ├── funding.md                # 奨学金・助成金・アシスタントシップ
    ├── publications.md           # リポジトリ・学位論文アーカイブ・OA方針
    ├── companies.md              # スタートアップ、スピンオフ、技術移転
    ├── alumni.md                 # 著名な卒業生と受賞歴
    ├── roadmap.md                # 高校生から研究者を目指すロードマップ
    └── resources.md              # 公式リンク、OCW、YouTube、ポッドキャスト
```

## 🇨🇳 中文

### 📖 项目简介

本仓库是一个面向全球顶尖研究型大学的**长期结构化研究数据库**。与一般的大学介绍不同,本项目专为**学生、研究人员和开发者**打造,系统性地整理以下信息,并附带可核实的来源:

- 本科与研究生申请流程
- 研究机会、实验室与教师信息
- 资助、奖学金与实习机会
- 校友、衍生企业与技术转移
- 面向未来研究者的成长路线图

所有内容均使用 Markdown 编写,优先引用**大学官方来源**,并遵循统一模板,以便未来实现跨校对比。

### 🗂️ 仓库结构

```text
universities/
└── <university-slug>/
    ├── overview.md               # 历史、校园、院系结构、排名、统计数据
    ├── admissions.md             # 本科与研究生申请要求
    ├── undergraduate_research.md # SURF、奖学金、独立研究项目
    ├── graduate_research.md      # 博士/硕士项目、导师匹配
    ├── laboratories.md           # 所有实验室(院系、方向、负责人、官网)
    ├── professors.md             # 现任教师名录
    ├── research_centers.md       # 研究所与研究中心
    ├── internships.md            # 校内外实习机会
    ├── funding.md                # 奖学金、助学金、助教/助研岗位
    ├── publications.md           # 论文库、学位论文档案、开放获取政策
    ├── companies.md              # 初创企业、衍生公司、技术转移
    ├── alumni.md                 # 知名校友及所获荣誉
    ├── roadmap.md                # 从高中到研究生涯的成长路线图
    └── resources.md              # 官方链接、开放课程、YouTube、播客
```
