# Caltech — Laboratories (Index)

`universities/caltech/laboratories/README.md`

This directory replaces a single flat `laboratories.md` because Caltech's lab/research-group count is too large to responsibly maintain in one file. Each file below corresponds to one division or option and follows the same table format:

```
| Laboratory / Research Group | Department | Research Areas | Director | Official Website |
```

For fields without traditional PI-directed physical labs (e.g., pure mathematics), the table uses "research group" in place of "laboratory" — see the note at the top of each such file.
